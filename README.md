# 🍽️ Menu Digital - Restaurant

Système de menu digital pour restaurant avec commande en ligne.

## 📱 Fonctionnalités

- Menu digital accessible par QR code
- Commande en ligne par table
- Panier en temps réel
- Administration des produits et catégories
- Suivi des commandes

## 🔗 Accès

- **Menu** : `https://simou-rayane.github.io/menu-dib/menu.html`
- **Administration** : `https://simou-rayane.github.io/menu-dib/admin.html`
- **Commandes** : `https://simou-rayane.github.io/menu-dib/orders.html`

## 🔧 Configuration

1. Créez un projet Firebase
2. Configurez les secrets GitHub :
   - `FIREBASE_API_KEY`
   - `FIREBASE_AUTH_DOMAIN`
   - `FIREBASE_DATABASE_URL`
   - `FIREBASE_PROJECT_ID`
   - `FIREBASE_STORAGE_BUCKET`
   - `FIREBASE_MESSAGING_SENDER_ID`
   - `FIREBASE_APP_ID`

## 📦 Déploiement

Le projet est automatiquement déployé sur GitHub Pages via GitHub Actions.

## 🛠️ Technologies

- Firebase Realtime Database
